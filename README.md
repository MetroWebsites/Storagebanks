## Setup

1. Clone the repository
2. Open `index.html` in a web browser
3. Or serve with any static file server

## Form Handling

All forms on the website use the `mailto:` protocol to open the user's default email client with pre-filled information. This ensures:
- No server-side processing needed
- User controls when emails are sent
- All form data is visible to the user before sending
- Works on all devices with email clients
